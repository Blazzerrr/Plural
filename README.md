# Plural 

Возвращает строку с множественным числом

## Установка
```
pip install pluralru
```

## Как использовать Plural
```python
from pluralru.pluralru import Plural

p = Plural('рубль', 'рубля', 'рублей')
result = p.plural_result('15')
print(result) # 15 рублей
```
